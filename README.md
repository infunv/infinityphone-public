# InfinityPhones Public Repository

Official public assets, configurations, and resource pack repository for the **InfinityPhones** Minecraft plugin, managed by **INFUNV STUDIO** (`infunv.ru`) and its Minecraft department (`mine.infunv.ru`).

---

## Repository Overview

This public repository serves as the central hub for hosting and distributing public assets associated with the InfinityPhones project:

* **Resource Pack Releases**: Compiled resource pack archives containing 3D models, textures, and custom GUI layouts are published directly under the [Releases](../../releases) section of this repository.
* **Automatic Resource Pack Delivery**: The InfinityPhones plugin automatically prompts and applies the resource pack upon a player's join if server resource packs are enabled and permitted by the client.
* **Default Configurations**: Default configuration files are provided in the root directory for reference. Localized Russian configurations are located in the `ru/` subdirectory.
